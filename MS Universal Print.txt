# Universal print ports used
Source: https://techcommunity.microsoft.com/t5/universal-print/universal-print-ports-used/m-p/3611491

  To use universal print the following endpoints must be reachable:

  *.print.microsoft.com
  *.microsoftonline.com
  *.azure.com
  *.msftauth.net
  go.microsoft.com
  aka.ms
